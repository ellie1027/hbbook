# 1. 설치

* npm이 먼저 설치되어 있어야 한다.

```shell
npm install -g @angular/cli
```

설치 후 프로젝트 폴더를 하나 만들고

```shell
ng new ang01
```

프로젝트 폴더로 옮긴 후

```shell
cd ang01
ng serve --open
```

서버를 실행한다.
